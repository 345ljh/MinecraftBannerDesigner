# Minecraft Banner Designer | MC旗帜编辑器
一个功能完整的 Minecraft 旗帜离线设计工具，基于 Python 和 PyQt5 开发，无需连接游戏即可设计和预览旗帜图案，并支持MC指令导出，旨在让大型旗帜图案设计、分享更加简单和高效。  
使用说明可参考```\images\hint```下图片，或在编辑器内打开“快捷键提示”界面。  

前代演示：https://www.bilibili.com/video/BV1KZ4y1R7t6  

## 更新日志  
1.添加大量快捷键，**支持在不使用鼠标的情况下完成所有必要操作**，并添加翻转、撤销/重做、设计搜索等实用功能。  
2.重构旗帜存储方式，在兼容前代文件的基础上更为精简（*前代不兼容当前的.banner文件*），且取消了6个染色步骤的上限，可以在单面旗帜上进行**任意**次染色，享受无限制的创作自由。  
3.全面优化UI布局，控件大小随窗口自适应调整；支持设计**实时预览、缩放与背景颜色自定义**。  
4.优化**生存模式染料计数器**，精确计算旗帜所需的染料数量（每个染色步骤为1个染料，每面旗帜为6个染料）。  
5.支持联网检查更新功能。  
6.移除了Herobrine。
